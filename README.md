#  AI Resume Analyzer

An **industry-level AI Resume Screening System** that analyzes resumes, predicts job roles, detects skill gaps, and provides intelligent feedback.

---

##  Features

* 🎯 Multi-role prediction (Top matching roles)
* 🏷️ Skill extraction from resume
* ❌ Missing skill detection
* 📊 Resume strength scoring
* 📄 Resume completeness analysis
* 🤖 AI-based feedback & suggestions
* 📤 Supports PDF, DOCX, JPG, PNG

---

##  Tech Stack

* Python
* Scikit-learn
* NLP (TF-IDF)
* Streamlit

## 🧪 How it Works

1. Upload resume
2. Extract text (PDF/DOCX/Image)
3. Clean & preprocess text
4. Predict role using ML model
5. Extract skills
6. Compare with required skills
7. Generate score & feedback
